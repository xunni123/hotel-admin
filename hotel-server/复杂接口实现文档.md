# Dashboard 仪表盘复杂接口实现文档

## 概述

本文档详细描述了酒店管理系统中**仪表盘模块**的复杂接口实现，包括：
- 入住率统计（周/月/半年）
- 收入趋势统计
- 数据聚合与转换

这些接口涉及批量查询、日期处理、数据转换等复杂逻辑。

## 二、入住率统计接口实现

### 接口说明

**接口路径**: `GET /dashboard/occupancy`

**功能**: 获取周/月/半年的入住率趋势数据，用于前端折线图展示

**响应数据结构**:

```java
public class OccupancyStats {
    private List<OccupancyData> week;    // 近7天数据
    private List<OccupancyData> month;   // 近30天数据
    private List<OccupancyData> halfYear; // 近180天数据

    public static class OccupancyData {
        private String date;          // 日期
        private double occupancyRate; // 入住率
    }
}
```

### 核心实现代码

**DashboardService.java**:

```java
@Service
public class DashboardService {

    @Autowired
    private DashboardMapper dashboardMapper;

    /**
     * 获取入住率统计数据
     */
    public OccupancyStats getOccupancyStats() {
        try {
            LocalDate today = LocalDate.now();
            
            OccupancyStats stats = new OccupancyStats();
            stats.setWeek(calculateOccupancyData(today, 7));        // 近7天
            stats.setMonth(calculateOccupancyData(today, 30));      // 近30天
            stats.setHalfYear(calculateOccupancyData(today, 180));  // 近180天
            
            return stats;
        } catch (Exception e) {
            System.err.println("Error calculating occupancy stats: " + e.getMessage());
            return new OccupancyStats();
        }
    }

    /**
     * 计算指定天数内的入住率数据
     * 
     * 优化点：使用批量查询替代N次单条查询，提升性能
     */
    private List<OccupancyStats.OccupancyData> calculateOccupancyData(LocalDate today, int days) {
        try {
            List<OccupancyStats.OccupancyData> dataList = new ArrayList<>();
            Integer totalRooms = dashboardMapper.countTotalRooms(); // 获取总房间数

            // 优化：使用批量查询一次性获取所有日期的数据，而不是循环查询N次
            List<Map<String, Object>> batchResult = dashboardMapper.countOccupiedRoomsBatch(days - 1);

            // 构建日期 -> 数量的映射表
            Map<String, Long> countByDate = new HashMap<>();
            for (Map<String, Object> row : batchResult) {
                Object dateObj = row.get("date");
                Object countObj = row.get("count");
                if (dateObj != null) {
                    String dateKey = dateObj.toString();
                    long count = countObj != null ? ((Number) countObj).longValue() : 0;
                    countByDate.put(dateKey, count);
                }
            }

            // 遍历所有日期，计算入住率
            for (int i = days - 1; i >= 0; i--) {
                LocalDate date = today.minusDays(i);
                String dateKey = date.toString();
                double occupancyRate = 0.0;
                
                // 计算入住率 = (在住房数 / 总房量) * 100，保留1位小数
                if (totalRooms != null && totalRooms > 0) {
                    Long occupiedRooms = countByDate.getOrDefault(dateKey, 0L);
                    occupancyRate = Math.round((double) occupiedRooms / totalRooms * 100 * 10) / 10.0;
                }
                
                OccupancyStats.OccupancyData data = new OccupancyStats.OccupancyData();
                data.setDate(dateKey);
                data.setOccupancyRate(occupancyRate);
                dataList.add(data);
            }

            return dataList;
        } catch (Exception e) {
            System.err.println("Error calculating occupancy data for " + days + " days: " + e.getMessage());
            return new ArrayList<>();
        }
    }
}
```

### Mapper 层实现

**DashboardMapper.java**:

```java
@Mapper
public interface DashboardMapper {

    /**
     * 获取总房间数
     */
    @Select("SELECT COUNT(*) FROM room WHERE is_deleted = 0")
    Integer countTotalRooms();

    /**
     * 批量查询指定天数内的在住房数
     * 
     * SQL优化：使用 GROUP BY 一次性返回所有日期的数据
     */
    @Select("SELECT DATE(checkin_time) AS date, COUNT(*) AS count " +
            "FROM room " +
            "WHERE checkin_time >= DATE_SUB(CURDATE(), INTERVAL #{days} DAY) " +
            "AND status = 'checkedIn' " +
            "AND is_deleted = 0 " +
            "GROUP BY DATE(checkin_time)")
    List<Map<String, Object>> countOccupiedRoomsBatch(@Param("days") int days);
}
```

### 性能优化说明

| 优化前 | 优化后 |
|--------|--------|
| 循环N次查询数据库 | 批量查询1次 |
| 时间复杂度 O(N) | 时间复杂度 O(1) |
| N次网络IO | 1次网络IO |

### 接口说明

**接口路径**: `GET /dashboard/revenue`

**响应数据结构**:

```java
    private List<BigDecimal> revpar;      // RevPAR列表
}
```

### 核心实现代码

**DashboardService.java**:

```java
/**
 * 获取收入统计数据
 * 
 * 数据处理：将行数据转换为列数据，适配ECharts图表格式
 */
public List<Revenue> getRevenue() {
    try {
        // 1. 查询原始数据（行格式：每行包含日期、收入、RevPAR）
        List<Map<String, Object>> rows = dashboardMapper.getRevenueRows();
        if (rows == null || rows.isEmpty()) {
            return new ArrayList<>();
        }

        // 2. 数据转换：行 -> 列
        List<String> dates = new ArrayList<>();
        List<BigDecimal> revenueList = new ArrayList<>();
        List<BigDecimal> revparList = new ArrayList<>();

        for (Map<String, Object> row : rows) {
            // 处理日期
            Object dateObj = row.get("date");
            dates.add(dateObj != null ? dateObj.toString() : "");

            // 处理收入（BigDecimal类型转换）
            Object revObj = row.get("revenue");
            revenueList.add(revObj != null ? new BigDecimal(revObj.toString()) : BigDecimal.ZERO);

            // 处理RevPAR
            Object revparObj = row.get("revpar");
            revparList.add(revparObj != null ? new BigDecimal(revparObj.toString()) : BigDecimal.ZERO);
        }

        // 3. 封装结果
        Revenue revenue = new Revenue();
        revenue.setDates(dates);
        revenue.setRevenue(revenueList);
        revenue.setRevpar(revparList);

        List<Revenue> result = new ArrayList<>();
        result.add(revenue);
        return result;
    } catch (Exception e) {
        System.err.println("Error getting revenue: " + e.getMessage());
        return new ArrayList<>();
    }
}
```

### Mapper 层实现

**DashboardMapper.java**:

```java
/**
 * 查询收入趋势数据
 * 
 * SQL说明：
 * - 按日期分组统计
 * - 计算每日总收入
 * - 计算RevPAR = 总收入 / 总房间数
 */
@Select("SELECT " +
        "  DATE(create_time) AS date, " +
        "  SUM(amount) AS revenue, " +
        "  SUM(amount) / (SELECT COUNT(*) FROM room WHERE is_deleted = 0) AS revpar " +
        "FROM financial_record " +
        "WHERE type = 'income' " +
        "AND is_deleted = 0 " +
        "GROUP BY DATE(create_time) " +
        "ORDER BY date")
List<Map<String, Object>> getRevenueRows();
```

### 数据转换说明

**原始数据（行格式）**:

| date | revenue | revpar |
|------|---------|--------|
| 2024-06-01 | 25000.00 | 250.00 |
| 2024-06-02 | 28000.00 | 280.00 |

**转换后（列格式，适配ECharts）**:

```json
{
  "dates": ["2024-06-01", "2024-06-02"],
  "revenue": [25000.00, 28000.00],
  "revpar": [250.00, 280.00]
}
```

## 四、仪表盘统计接口实现

### 接口说明

**接口路径**: `GET /dashboard/stats`

**功能**: 获取核心业务指标：总房量、今日入住、今日退房、空房数、入住率、今日收入

### 核心实现代码

**DashboardService.java**:

```java
/**
 * 获取仪表盘统计数据
 * 
 * 包含逻辑：
 * 1. 查询多个统计指标
 * 2. 业务公式计算（空房数、入住率）
 * 3. 浮点数精度处理（BigDecimal）
 */
public DashboardStats getDashboardStats() {
    LocalDate today = LocalDate.now();

    // 1. 查询基础指标
    Integer totalRooms = dashboardMapper.countTotalRooms();
    Integer todayCheckIns = dashboardMapper.countTodayCheckIns(today);
    Integer todayCheckOuts = dashboardMapper.countTodayCheckOuts(today);
    Integer occupiedRooms = dashboardMapper.countOccupiedRooms(today);
    BigDecimal todayRevenue = dashboardMapper.sumTodayRevenue(today);

    // 2. 计算空房数 = 总房量 - 在住房数
    Integer emptyRooms = 0;
    if (totalRooms != null && occupiedRooms != null) {
        emptyRooms = totalRooms - occupiedRooms;
    }

    // 3. 计算入住率 = (在住房数 / 总房量) * 100
    // 使用BigDecimal保证精度，保留1位小数，四舍五入
    BigDecimal occupancyRate = BigDecimal.ZERO;
    if (totalRooms != null && totalRooms > 0 && occupiedRooms != null) {
        occupancyRate = BigDecimal.valueOf(occupiedRooms)
                .multiply(BigDecimal.valueOf(100))
                .divide(BigDecimal.valueOf(totalRooms), 1, RoundingMode.HALF_UP);
    }

    // 4. 封装结果
    DashboardStats stats = new DashboardStats();
    stats.setTotalRooms(totalRooms);
    stats.setTodayCheckIns(todayCheckIns);
    stats.setTodayCheckOuts(todayCheckOuts);
    stats.setEmptyRooms(emptyRooms);
    stats.setOccupancyRate(occupancyRate);
    stats.setTodayRevenue(todayRevenue);

    return stats;
}
```

**DashboardMapper.java**:

## 六、关键技术点总结

### 1. 批量查询优化

**问题**: 以前需要循环查询N次数据库获取N天的数据

**解决方案**: 使用 `GROUP BY` 一次性查询所有数据，然后在内存中处理

**性能提升**:
- 网络IO: N次 → 1次
- 数据库压力: N次查询 → 1次查询

### 2. 数据格式转换

**问题**: 数据库返回行格式，但前端ECharts需要列格式

**解决方案**:
- 原始数据: `List<Map<String, Object>>`（行格式）
- 目标数据: `{ dates: [], revenue: [], revpar: [] }`（列格式）
- 转换逻辑: 遍历行数据，分别填充到不同的列表

### 3. BigDecimal 精度处理

**问题**: 浮点数计算存在精度问题

**解决方案**:
- 使用 `BigDecimal` 替代 `double/float`
- 使用 `setScale(1, RoundingMode.HALF_UP)` 设置精度和舍入方式
- 收入计算: 保留2位小数
- 入住率计算: 保留1位小数

### 4. 异常处理与降级

**问题**: 数据查询可能失败，导致接口500

**解决方案**:
```java
try {
    // 业务逻辑
} catch (Exception e) {
    System.err.println("Error: " + e.getMessage());
    return new ArrayList<>(); // 返回空数据而非抛出异常
}
```

### 5. 日期处理

**问题**: 需要处理相对日期（近7天、近30天）

**解决方案**:
```java
LocalDate today = LocalDate.now();
LocalDate date = today.minusDays(i); // 往前推i天
```

## 九、扩展建议

### 1. 缓存优化

```java
public List<OccupancyStats.OccupancyData> calculateOccupancyData(...) {
```

### 2. 异步查询

对于多个独立的统计指标，可以并行查询：

```java
CompletableFuture<Integer> roomsFuture = CompletableFuture.supplyAsync(() -> dashboardMapper.countTotalRooms());
CompletableFuture<BigDecimal> revenueFuture = CompletableFuture.supplyAsync(() -> dashboardMapper.sumTodayRevenue(today));

CompletableFuture.allOf(roomsFuture, revenueFuture).join();
```

### 3. 数据预计算

对于历史数据，可以每天凌晨预计算并存入统计表，提升查询速度。
